# Демо контент-пайплайна Marketing OS

Эта демо-страница находится в процессе подготовки.

Здесь будет пример того, как голос, заметка или сырая идея проходят путь до SEO-статьи, карусели, публикации и архива.

## Что будет показано

```text
голос / заметка / тренд
→ транскрипт
→ смысловой план
→ SEO-черновик
→ редактура
→ карусель
→ публикация
→ архив в vault
→ повторное использование
```

## Зачем это нужно

Marketing OS не должна выглядеть как «еще один генератор текстов».

Демо должно показать главное: контент становится наблюдаемым процессом. У каждого материала есть вход, статус, следующий шаг, публикация и место в архиве.

## Черновая структура будущего демо

### 1. Вход

Будет показан один входной материал:

- голосовая заметка;
- фрагмент разговора;
- черновая мысль;
- тема из SEO-кластера.

### 2. Преобразование

Будет видно, как материал проходит этапы:

- расшифровка;
- выделение тезисов;
- сбор плана;
- генерация черновика;
- редакторская проверка;
- адаптация под формат.

### 3. Артефакты

На выходе появляются:

- статья;
- короткий пост;
- карусель;
- карточка в системе;
- ссылка на публикацию;
- заметка в vault.

### 4. Контроль качества

Демо должно показать, где человек остается в контуре:

- выбор смысла;
- проверка фактов;
- тон;
- финальное решение о публикации;
- связь с позиционированием.
