# Демо графа рекомендаций Recca

Эта демо-страница находится в процессе подготовки.

Здесь будет пример того, как Recca показывает цепочку рекомендаций, партнерских связей, событий, ролей и вклада.

## Что будет показано

```text
участник
→ рекомендация
→ переданный контекст
→ встреча
→ заявка
→ сделка
→ роли участников
→ вес вклада
→ граф рекомендаций
```

## Почему это важно

В партнерских продажах и сообществах ценность часто создается не последним кликом.

Кто-то познакомил.  
Кто-то объяснил контекст.  
Кто-то усилил рекомендацию.  
Кто-то сопровождал сделку.  
Кто-то вернул человека в разговор через месяц.

Обычная реферальная программа часто видит только промокод или финальную заявку. Recca должна показать цепочку.

## Черновая структура будущего демо

### 1. Участники

В примере будут обезличенные участники:

```text
A — эксперт
B — партнер
C — потенциальный клиент
D — участник сообщества
```

### 2. События

Будут показаны события:

- познакомил;
- порекомендовал;
- передал контекст;
- пригласил на встречу;
- отправил оффер;
- довел до заявки;
- помог закрыть.

### 3. Роли

Каждый участник получает роль в событии:

- инициатор;
- рекомендатель;
- проводник;
- эксперт;
- сопровождающий;
- покупатель.

### 4. Вес вклада

Демо покажет, как можно не сводить вклад к одному человеку.

```text
событие → роль → вес → вклад
```

### 5. Граф

Финальный результат — не таблица ради таблицы, а карта коммуникационных связей, которые реально создают возможности.
