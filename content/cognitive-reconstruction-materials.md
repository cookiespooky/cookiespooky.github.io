# Материалы для анализа заметок

Загрузите ваш [[cognitive-reconstruction-brief|архив с заметками]] в чат с любой нейросетью и используйте промпты ниже для базового анализа.
## Этап 1. Построение карты знаний

Цель: понять, что вообще лежит в архиве.
### Промпт

```
Проанализируй граф заметок пользователя, включая вложенные папки.

Построй карту знаний и выдели:

- концепции
- методы
- инструменты
- модели
- гипотезы
- наблюдения
- повторяющиеся темы

Создай граф знаний:

- узлы = концепции
- связи = явные и косвенные упоминания
- вес связи оценивай по:
    - частоте совместного появления
    - контекстной близости
    - причинно-следственным отношениям
    - отношению часть-целое

Верни:

- список узлов
- список связей
- основные кластеры
- центральные концепции
- периферийные концепции
```

---
## Этап 2. Когнитивная реконструкция автора

Теперь анализируем не знания, а мышление.
### Промпт

```
Используя граф знаний и архив заметок:

Определи особенности мышления автора.

Проанализируй:

- через какие концепции он чаще объясняет мир
- какие концепции являются первичными и вторичными
- какие паттерны повторяются между разными областями
- какие метафоры используются регулярно
- какие причинно-следственные модели встречаются чаще всего

Построй:

- карту мышления
- карту абстракций
- карту объяснительных моделей

Выдели основные линзы восприятия реальности.
```

---
## Этап 3. Поиск когнитивных пустот

Только после когнитивной модели. Иначе пустоты будут галлюцинацией.
### Промпт

```
Используя граф знаний и карту мышления автора:

Найди когнитивные пустоты.

Определи:

- концепции, которые логично должны присутствовать, но отсутствуют
- слабосвязанные области
- недостроенные объяснительные цепочки
- концепции, вокруг которых есть высокая плотность заметок, но отсутствует объединяющая модель
- области с высокой вероятностью дальнейшего исследования

Для каждой пустоты объясни:

- почему она возникла
- какие связи на нее указывают
- что изменится при ее заполнении
```

---
## Этап 4. Поиск скрытых связей

### Промпт

```
Используя граф знаний:

Найди связи между кластерами, которые автор явно не описывает.

Ищи:

- структурные аналогии
- одинаковые механизмы под разными названиями
- повторяющиеся паттерны
- концепции-мосты
- фрактальные структуры

Оцени силу связи по шкале:

- слабая
- вероятная
- сильная
- фундаментальная

Для каждой связи объясни механизм.
```
